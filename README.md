## Local Machine Installation
##### Prerequisites
* Node v8.5.0^
* MongoDB (mongod) running on local machine

#### OSX/Windows
1.  Rename ".env.example" to ".env" and change the placeholder values to suit your environment.
2.	Run “npm install”
3.	Run "npm start"
